# FlexEra Exercise AI API

Real-time exercise validation using YOLOv8 pose detection for the FlexEra physiotherapy app.

## Endpoints

- `GET /docs` — Swagger UI
- `GET /health` — Health check
- `POST /process` — Process a frame
- `WS /ws` — WebSocket for real-time pose detection
